**Simulation Rules and Agent Behaviors**
* [cite_start]"Agents should increase their communication success rate by having their token weight increasing"[cite: 37].
* [cite_start]"the number of monolingual agents should always be superior to bilinguals in both worlds (we can start with a 3/2 ratio :)"[cite: 38].
* [cite_start]"communication success is modulated by the probability for an agent to meet another"[cite: 39].
* [cite_start]"Therefore, if the probability for a bilingual agent (A2) to talk to a monolingual agent (A1) is high, the token weight should increases"[cite: 40].
* [cite_start]"Same apply for a bilingual agent (A2) to talk to another bilingual"[cite: 41].
* [cite_start]"Agents will try to match their guesses when communicating color terms"[cite: 42].
* [cite_start]"If the matching is wrong the agent will find another peer to initiate another communication"[cite: 43].
* [cite_start]"On the contrary, if their guesses converge on the same meaning, the weight of the token increases"[cite: 44].
* [cite_start]"Eventually, agents will try to communicate only to agents that match most of their tokens weight"[cite: 45].
* [cite_start]"Bilinguals in World1 will always use the Language 1 to communicate with monolingual agents"[cite: 46].
* [cite_start]"Bilinguals in World2 will communicate in both Language 1 and Language 2 with native hosts and immigrants"[cite: 47].
